余剰・課税ビジュアライザー

使い方

概要

需要曲線 P = a − b·Q と供給曲線 P = c + d·Q のもとで、市場均衡における消費者余剰・生産者余剰を表示する。従量税 t を加えると税収と死荷重損失も可視化される。

パラメータ

  • 需要切片 a: 価格軸の切片（最大支払意思額に対応）
  • 需要の傾き b: 値が大きいほど急（垂直に近いほど必需財寄り）
  • 供給切片 c: 最低供給価格
  • 供給の傾き d: 値が大きいほど急
  • 従量税 t: 取引 1 単位あたりに課される税．ここでは消費者に課される

図の見方

  • 消費者余剰 (CS)
  • 生産者余剰 (PS)
  • 税収（t > 0 のとき）
  • 死荷重損失 (DWL)（t > 0 のとき）
  • 課税時には消費者価格 P_D（税込）と生産者価格 P_S（税抜）が縦軸に併記される

試してみよう

  • t を 0 から徐々に増やすと、CS と PS が縮小し、代わりに税収と DWL が現れる
  • 需要の傾き b を急にする（必需財寄り）と、同じ t でも DWL は相対的に小さくなる
  • c ≥ a になると均衡が存在せず、警告が表示される
  • t を大きくしすぎると課税により市場が消滅する
